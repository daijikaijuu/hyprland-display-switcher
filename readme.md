## Installation

### Prerequisites

- Hyprland window manager
- Rust toolchain (for building from source)

### Building from Source

```bash
git clone https://github.com/daijikaijuu/hyprland-display-switcher.git
cd hyprland-display-switcher
cargo build --release
```
